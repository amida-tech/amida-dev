Identity

Before any PHIX user can fully take advantage of the system, they need to have their profile and account validated. The identity endpoints allows you to see if a user is validated as well as pull the identity of a PHIX user. Identity endpoints uses a token received as part of any users profile, when pulled via account endpoints.

  • /identity/account (GET) - Takes an input token and displays the demographic information for identity verification.
  • /identity/validate (PUT) - Takes an input token with a boolean value to indicate whether or not an individual is verified.

You can access, and play with all the identity methods, using the PHIX interactive documentation.
