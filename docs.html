PHIX API Docs

These are working API docs for a sandbox PHIX environment, to show you what is possible with PHIX.

To start using this documentation, begin with the Account /login method, authenticate, then you will be ble to use the rest of the methods.

Authentication

The PHIX application andn API uses cookies for authentication, this documentation uses API key(s) to access the sandbox API. Visit authentication page for more information.
