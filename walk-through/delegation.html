Delegation

In addition to control which providers have access to their personal health information exchange, PHIX users can allow other users to have access to their accounts. All delegation endpoints provide you with the resources you need to create, view and remove requests for access to individual PHIX user personal health information accounts by other user.

  • /delegation (PUT) - Establishes access to the authenticated user for another user account.
  • /delegation (DELETE) - Removes access to the authenticated user for another user account that already has access.
  • /delegation/received (GET) - Retrieve a list of delegations granted to the authenticated user from other accounts.
  • /delegation/granted (GET) - Retrieve a list of delegations granted by the authenticateduser to other accounts.

You can access, and play with all the delegation methods, using the PHIX interactive documentation.
