What Is The Technology Behind PHIX?

PHIX begins with an API first strategy, deploying a simple API using Node.js, then building the PHIX individual and clinician web application interface on top of the APIs, using Node.js.

PHIX employs the latest approaches to deliver APIs and web applications using:

  • Node.js - Node.js is a platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications.
  • MongoDB - MongoDB is a cross-platform document-oriented database system, which is classified as a NoSQL database.
