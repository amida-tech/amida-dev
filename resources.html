Amida Dev Resources

This is the resources area, where you can find all the resources you will need to be successful developing and deploying around the PHIX platform.

Technology

  • Node.js - A platform built on Chrome's JavaScript runtime for easily building fast, scalable network applications.
  • NPM - A package manager for Node.js.
  • MongoDB - A cross-platform document-oriented database system, which is classified as a NoSQL database.
  • Grunt - A Node.js task runner for simplifying chores.

Documentation

  • Walk Through - A wizard walk through all elements of the PHIX platform.
  • Interactive Docs - Interactive documentation for making calls against the PHIX API to explore how it works.

Standards

  • Direct Project Security Overview - The Direct Project is a set of specifications and service descriptions that, when implemented within a strong policy framework, enables simple, secure point-to-point electronic messages between health care participants.
  • Blue Button / CCDA - An electronic health record keeps a digital record of clinical information about a particular individual. With the amount of variability among Health IT systems, it is important to have a consistent set of health information for each patient.
