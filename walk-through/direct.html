Direct

A significant aspect of each PHIX userâ€™s account is the ability to communicate with their healthcare provider(s). PHIX provides a mailbox for each user to communicate directly with their designated provider(s) using healthcare IT DIRECT standards. The direct endpoints provide you with the resources you need to send, receive and manage messages in the users mailbox.

  • /direct/message (PUT) - Creates a new message for a user to a provider in the system.
  • /direct/message (POST) - Updates an existing message to a status of read or archived.
  • /direct/message (DELETE) - Removes a message in the inbox by setting the archive flag to true.
  • /direct/inbox (GET) - Retrieves the content of the authenticated users inbox.
  • /direct/outbox (GET) - Retrieves the content of the authenticated users outbox.

You can access, and play with all the direct methods, using the PHIX interactive documentation.
