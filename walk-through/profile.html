Profile

Each PHIX user has an account in the system, which you use account endpoints to manage, but additionally each user has a profile which is the personal view of the user that is not associated with account access. Profile endpoints contains everything you need to know including contact information and demographics.

  • /profile (GET) - Retrieves the profile for the authenticated user.
  • /profile (PUT) - Creates a new profile for authenticated users.
  • /profile (POST) - Updates the profile for authenticated user.

You can access, and play with all the profile methods, using the PHIX interactive documentation.
