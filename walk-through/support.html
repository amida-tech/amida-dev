Code

This site is all about supporting PHIX deployments and the community of developer and technology provers who are deploying health care information exchanges. If there is anything you want to see in the way of resources, features or other supporting elements, submit your thoughts via the Github issues page.

This site runs on Github pages, and is supported as a living, open Github repository. The roadmap for all future PHIX developer work will occur here, so if you want your ideas, needs addressed as part of the roadmap, make sure it is added to Github.
