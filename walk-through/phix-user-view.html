PHIX User View

There are two main views of the PHIX application, the first is the individual user view. This is the view of the application that the end-user users to manage their own personal health information.

This view is the central view of the application, and as a developer and technology solution provider this is the view you need to focus on, remembering that this is about delivering a secure and solid solution for individuals to use with confidence.
