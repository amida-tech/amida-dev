Where Is PHIX Installed?

PHIX is an open source, downloadable solution. It can be installed on any server that can run Node.js and has MongoDB installed. This allows PHIX to deployed anywhere that a provider or individual wants, as well as opens up the opportunity for technology providers to step up and help deliver personal information exchanges for users.

Like other open source solutions, PHIX is meant to enable any individual or company to leverage the investment into PHIX, and join a community of developers and technology solution providers to deliver a consistent personal health information solution on any server or cloud environment.
