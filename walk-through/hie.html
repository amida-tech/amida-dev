Health Information Exchange (HIE)

The HIE or Health Information Exchange endpoint provides API access to a PHIX users health information, for use by healthcare providers. The HIE endpoints allows you to lookup, add, update and delete a users healthcare information.

  • /hie/lookup (POST) - Make a request to access a PHIX users healthcare information.
  • /hie (GET) - Looks up an outstanding request for PHIX users healthcare information.
  • /hie (PUT) - Creates a new request for a PHIX users healthcare information for a provider.
  • /hie (DELETE) - Removes a pending request for a PHIoX users healthcare information from a provider.

You can access, and play with all the hie methods, using the PHIX interactive documentation.
