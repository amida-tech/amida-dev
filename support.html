Amida Dev Support

This Amida Dev site is developed to run within a Github respository, published using Github pages. So everything about this community is Github centered.

Sticking with the trend of using Github, all support is done using Github issues. When you have any questions, comments or issues you just need to visit the Github issues page, and submit your request.

Use the icons to the right, they will take you to the issue page.
